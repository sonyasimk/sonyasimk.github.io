CompileBPV

A compiler for a toy functional language that uses Levy's Call-By-Push-Value as an internal representation. This project was completed as the final project for the course 15-791: Advanced Topics in Foundations of Programming Languages at CMU, and is based on a talk given by Max S. New.

ATPL Lecture Notes

Lecture notes on using phase distinctions and language constructs in a programming language to express equality between terms with respect to cost. These notes were written for the course 15-791: Advanced Topics in Foundations of Programming Languages at CMU.
